FlexDock was formerly known as the Eleritec docking framework. We are currently looking for developers to improve upon the ground work laid by Eleritec. The documentation for Flexdock .3 (Eleritec) can be found at: http://www.eleritec.net/projects/docking.html

Initial work as been done by Claudio Romano to provide support in Spring Rich Client for FlexDock. It is in the Spring Rich roadmap to offer official integration with docking frameworks through a common set of windowing abstractions. The basic abstractions are in place and people are extending them as in Claudio's case with FlexDock.
